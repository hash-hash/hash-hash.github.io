Information Theory and Coding Theory

Author: John Doe Date: July 11, 2022  18:29:06 Category: notes

暑期课程笔记

hamming code

就利用奇偶校验实现一比特纠错功能，迟到一小时看的时候老师画了个文氏图在解释，主要方法就是$2^i$的位置为校验位，其加入满足在idx二进制下的第i位=1的位置求和为偶，若出现一比特的错误，将导致idx的二进制下对应1的位置的sum为奇数，可通过奇偶纠错找到错码位置

prefix code

根据编码字符出现的频率，构建对应的编码规则，使得平均码字长度尽可能短

shannon entropy

用信息熵$H(p_1,p_2,…,p_n)$量化信息的平均不确定性

性质

  1. $H(p_1,p_2,…,p_n)$在$p_i$上连续
  2. $H(\frac{1}{2},\frac{1}{2})=1$
  3. $H(\frac{1}{n},…,\frac{1}{n})$随n增加而增加
  4. grouping axiom

$H(p_1,…,p_n)=\sum_{i=1}^mp_ilog(\frac{1}{p_i})$

proof:

记$f(m)=H(\frac{1}{m},…,\frac{1}{m})$

先证明引理$f(m^k)=kf(m)$

记$p_i=\frac{q_i}{M}$

利用分组原则得

$H(p_1,…,p_n)=log_M-\sum_{i=1}^mp_ilog(q_i)$

$=\sum_{i=1}^mp_i[logM-logq_i]=\sum_{i=1}^mp_ilog(\frac{1}{p_i})$

Markov chains

$P_{X_1^n}(x_1^n)=P_{X_1}(x_1)P_{X_2|X_1}(x_2|x_1)…$

即t+1的状态与t状态有关

有关公式

$P_{XYZ}(x,y,z)=P_Y(y)P_{XZ|Y}(x,z|y)$

$E[g(X,Y)]=\sum_{x∈\mathcal{X},y∈\mathcal{Y}}P_{XY}(x,y)g(x,y)$

Convex set

见到了一种有意思的定义

define a random variable V with $P_V(v_i)=p_i$ , satisfy $\sum p_iv_i=E[V]∈V$

接着在凸集基础上讲了下琴生

typical sequence

对典型集的定义是，对于权值给定p的数在n长序列中出现pn次，下面计算出现概率

$P_{X_1^m}(x_1^m)=\prod_{i=1}^mP_X(x_i)≈\prod_{x∈\mathcal{X}}P_X(x)^{P_X(x)m}$

$=2^{-m\sum P_X(x)log\frac{1}{P_X(x)}}=2^{-mH(x)}$

所以典型集的势$≈2^{mH(x)}$

补充了H(X)的性质

$0≤H(X)≤log|\mathcal{X}|$

其中，$H(X)=0 <=>$ X 是确定分布

$H(X)=log|\mathcal{X}| <=>$ X 为平均分布

直接加权琴生套取等条件即可

Joint Entropy

$H(X,Y)=\sum_{x,y}P_{XY}(x,y)log\frac{1}{P_{XY}(x,y)}$

$0≤H(x,y)≤log(|x||y|)$

$H(X,Y)=E[log\frac{1}{P_{XY}(x,y)}]=E[log\frac{1}{P_X(x)}]+E[log\frac{1}{logP_{Y|X}(y|x)}]=H(X)+H(Y|X)$

Conditional Entropy

$H(X,Y)=H(X)+H(Y|X)=H(Y)+H(X|Y)$

结论 $0≤H(Y|X)≤H(Y)$

$H(Y|X)=0 <=>$Y is deterministic func of X

$H(Y|X)=H(Y) <=>$X , Y is independent

means that conditioning doesn’t increase the entropy

Multiple entropy

$H(X_1,X_2,…,X_m)=\sum P_{X_1^m}(x_1^m)log\frac{1}{P_{X_1^m}(x_1^m)}$

Mutual Information

$H(X)+H(Y|X)=H(Y)+H(X|Y)$

$=>H(X)-H(X|Y)=H(Y)-H(Y|X)$

define $I(X;Y)=H(X)-H(X|Y)=H(Y)-H(Y|X)$

称为X，Y的互信息

Conditional Mutual Information

$I(X;Y|Z)=H(X|Z)-H(X|Y,Z)=H(Y|Z)-H(Y|X,Z)$

不存在多元的互信息，但是可以推广到数组间的

$I(X_1^m;Y)=I(X_1;Y)+I(X_2;Y|X_1)+…+I(X_m;Y|X^{m-1})$

proof $I(X;Y|Z)=\sum P_Z(z)I(X;Y|Z=z)$

$RHS=\sum(P_Z(z)H(X|Z=z))-(P_Z(z)H(X|Y,Z=z))$

$=\sum (H(X|Z)-H(X|Y,Z))=I(X;Y|Z)$

韦恩图的表示

个人感觉就是互信息越小两变量越疏离，越大两变量越紧密

Data Processing Inequality

for markov chain

$X->Y->Z$

$I(X;Y)≥I(X;Z)$

结论

$I(X;Y)$ is a concave func of Px for fixed $P_{Y|X}$，这个常在capacity of channel上使用

$I(X:Y)$ is a convex func of $P_{Y|X}$ for fixed Px，这个常在Rate distortion theory上使用

Fano’s Inequality

$X->Y->\hat{X}$，$P_e=P[\hat{X}≠X]$

proof $H(X|Y)≤H_{bin}(P_e)+P_elog(|\mathcal{X}|-1)$

这里记录老师给的一种clever trick

定义$E=\begin{cases}0，\hat{X}=X\\1，\hat{X}≠X\end{cases}$

$\begin{cases}P_E(1)=P_e\\P_E(0)=1-P_e\end{cases}$

∵$H(X,E|\hat{X})=H(X|\hat{X})+H(E|X,\hat{X})=H(E|\hat{X})+H(X|E,\hat{X})$

我们有

  1. $H(E|X,\hat{X})=0$
  2. $H(E|\hat{X})≤H(E)=H_{bin}(P_e)~(H_{bin}(P)=plog\frac{1}{p}+(1-p)log\frac{1}{1-p})$
  3. $H(X|\hat{X},E)=P_E(0)H(X|\hat{X},E=0)+P_E(1)H(X|\hat{X},E=1)≤P_elog(|\mathcal{X}|-1)$

综上可得 $H(X|Y)≤H_{bin}(P_e)+P_elog(|\mathcal{X}|-1)$

$=>P_e≥\frac{H(X|Y)-1}{log(|\mathcal{X}|-1)}≥\frac{H(X|Y)-1}{log|\mathcal{X}|}$

对于一个非负随机变量V，有以下结论

Markov Inequality

if V is non-negative ,

Then ∀ e>0 , $P[V≥e]≤\frac{E[V]}{e}$

Chebyshev Inequality

a random variable V with $E[V]=μ_v$ and $var[V]=σ_v^2$ .

Then $P[|v-μ_v|≥ε]≤\frac{σ_v^2}{ε^2}$

Weak Law of Large Numbers

if $v_1,v_2,…,v_n$ are i.i.d random variables with finite , $E[v_i]=μ_v$ and $var[v_i]=σ_v^2$

defining $\hat{V}^{(n)}=\frac{1}{n}\sum_{i=1}^nv_i$

then $P[|\hat{V}^{(n)}-μ_v|≥ε]≤\frac{σ_v^2}{nε^2}$

Asymptotic Equipartition Property

AEP

若$X_1,X_2,…,X_n$为i.i.d~p(x)

则$-\frac{1}{n}logp(X_1,…,X_n)->H(x)$依概率

典型集的相关性质

  1. $(x_1,x_2,…,x_n)∈A_ε^{(n)}$，则$H(x)-ε≤-\frac{1}{n}logp(x_1,…,x_n)≤H(x)+ε$
  2. 当n充分大时，$Pr{A_ε^{(n)}}>1-ε$
  3. $|A_ε^{(n)}|≤2^{n(H(x)+ε)}$
  4. 当n充分大时，$|A_ε^{(n)}|≥(1-ε)2^{n(H(x)-ε)}$

对于非均等分布的X，典型集的概率$≈2^{-m(H(x)±ε)}$，平均概率约为1

Shannon’s Theorem

这个理论挺重要的，大概是说的对随机变量的编码仅考虑在典型集中能做到长度短且误差小

Let’s $x_1,x_2,…$ be i.i.d random variables on a finite $\mathcal{X}$. Then for any ε>0

we can encode in binary form these variables using a number of bits per symbol H(x)+ε with

probability of error small as n grows

Conversely, if we encode using a number of bits per symbol less than H(x)-ε’, then as n grows the

probability of error goes to 1
