A Decade of Lattice Cryptography

Author: John Doe Date: September 30, 2022  10:05:23 Category: Cryptography

paper

基于格的密码系统的一些特性

  • Conjectured security against quantun attacks
  • Algorithmic simplicity, efficiency, and parallelism
  • Strong security guarantees from worst-case hardness
  • Constructions of versatile and powful cryptographic objects

The Dual lattice

the dual lattice of a lattice $L⊆R^n$ is defined as $L^※:={w:<w,L>⊆Z}$

Orthogonal lattice

The orthogonal lattice of a lattice $L∈Z^m$ is $L^⊥={v∈Z^m|∀b∈L,<v,b>=0}$

Computational Problems

  1. Shortest Vector Problem

    Given an arbitrary basis B of some lattice $L=L(B)$, find a shortest nonzero lattice vector.

  2. Approximate Shortest Vector Problem

    Given a basis B of an n-dimensional lattice $L=L(B)$, find a nonzero vector $v∈L$ for which $‖v‖≤γ(n)·\lambda_1(L)$ .

  3. Decisional Approximate SVP

    Fixed d,whether exit a vector $v∈L$ for which $γ(n)·‖v‖≤d$

  4. Approximate Shortest Independent Vectors Problem

    Given a basis B of a full-rank n-dimensional lattice $L=L(B)$, output a ser $S=\left{s_i\right}⊆L$ of n linearly independent lattice vector where $‖s_i‖≤γ(n)·\lambda_n(L)$ for all i.

  5. Bounded Distance Decoding Problem

    Given a basis B of an n-dimensional lattice $L=L(B)$ and a target point $t∈R^n$ with the guarantee that $dist(t,L)<d$, find the unique lattice vector $v∈L$ such that $‖t-v‖<d$.

Gaussians

For any positive integer n and real s>0, which is taken to be s=1 when omitted, define the Gaussian function $\rho_s:R^n→R^+$ of parameter s as $\rho_s(x):=e^{-\pi‖x‖^2/s^2}$

the continuous Gaussian distribution $D_s$ of parameters over $R^n$ is defined to have probability density function proportional to $\rho_s$, $f(\vec{x}):=\rho_s(\vec{x})/s^n$

高斯分布的话其实就是提供的高维度上的一种采样方式，和正态分布类似

Early Results

Ajtai-Dwork Encryption

挺有意思的一个方案，基于HHP问题所构建，但是其问题在于要保证在相关攻击下的安全性需要产生较大规模的密钥，效率低下

Hidden hyperplanes problem

Let s ∈ $R^n$ be a secret, random short vector. The goal is to find s, given many points $y_i∈R^n$ such that $<s,y_i>$ is close to an integer, i.e., $<s,y_i>≈0(mod~1)$. In the other words, each $y_i$ is close to one of the parallel (n-1)-dimensional hyperplanes $H_j={z∈R^n:<s,z>=j}$, j∈Z.

  1. 选取私钥s∈$R^n$
  2. 随机选取$v_i∈{v|<v,s>=Z}$
  3. 将$S={v_i}$作为公钥，s作为私钥

enc

对于明文的每个比特若为0则输出$R^n$中的随机向量，若为1则输出S中的若干元素和

通过将输出与私钥s点乘若为整数则输出1否则输出0

基于HHP所以从公钥pk推出私钥s是困难的

NTRU

NTRU的quotient ring一开始是选的$x^n-1$后面改成了分圆多项式$x^n+1$，但是$x^n+1$有比较好的代数结构，所以最终出了个NTRU Prime，调整了NTRU的相关结构使得它更加安全，一个改动就是quotient ring改成了$x^n-x+1$

using large Galois groups instead of cyclotomics

相关可以参考这篇paper，算是比较新的了

另外一个就是关于NTRU的格构造，实际上NTRU格是较为难解的格了，由于私钥的循环均是短向量，导致可出现大量较短的线性组合

而我们知道想要LLL的算法成功实施是希望我们要的短向量与次短向量出现较大的gap，而这里大量线性组合出的较短向量都在LLL的规约范围内，这就导致了我们所规约出的并不是密钥或等效密钥

实验的时候发现似乎q较小或者格子维数过高时都无法得到理想的规约结果

Modern Foundations

SIS

def Given m uniformly random vector $a_i∈Z_q^n$, forming the columns of a matrix $A∈Z_q^{n×m}$, find a nonzero integer vector $z∈Z^m$ of norm $||z||≤β$ such that Az=0∈$Z_q^n$

LWE

Ring-SIS

与SIS类似，向量换多项式

给定一个多项式向量A求短的多项式向量X满足$A·X=0(R_q)$

当$R=\frac{Z[x]}{x^n-1}$时$X=(1+x+x^2+…+x^{n-1},0,…,0)$有$\frac{1}{q}$的概率能满足要求，所以Ring-SIS的商环一般考虑分圆多项式

对于多项式一般考虑构造ideal lattice进行约化
