The Dual lattice

the dual lattice of a lattice $L⊆R^n$ is defined as $L^※:=\left{w:<w,L>⊆Z\right}$

Orthogonal lattice

The orthogonal lattice of a lattice $L∈Z^m$ is $L^⊥=\left{v∈Z^m|∀b∈L,<v,b>=0\right}$

Computational Problems

Shortest Vector Problem

Given an arbitrary basis B of some lattice $L=L(B)$, find a shortest nonzero lattice vector.

Approximate Shortest Vector Problem

Given a basis B of an n-dimensional lattice $L=L(B)$, find a nonzero vector $v∈L$ for which $‖v‖≤γ(n)·\lambda_1(L)$ .

Decisional Approximate SVP

Fixed d,whether exit a vector $v∈L$ for which $γ(n)·‖v‖≤d$

Approximate Shortest Independent Vectors Problem

Given a basis B of a full-rank n-dimensional lattice $L=L(B)$, output a ser $S=\left{s_i\right}⊆L$ of n linearly independent lattice vector where $‖s_i‖≤γ(n)·\lambda_n(L)$ for all i.

Bounded Distance Decoding Problem

Given a basis B of an n-dimensional lattice $L=L(B)$ and a target point $t∈R^n$ with the guarantee that $dist(t,L)<d$, find the unique lattice vector $v∈L$ such that $‖t-v‖<d$.

Gaussians

For any positive integer n and real s>0, which is taken to be s=1 when omitted, define the Gaussian function $\rho_s:R^n→R^+$ of parameter s as $\rho_s(x):=e^{-\pi‖x‖^2/s^2}$

the continuous Gaussian distribution $D_s$ of parameters over $R^n$ is defined to have probability density function proportional to $\rho_s$, $f(\vec{x}):=\rho_s(\vec{x})/s^n$
